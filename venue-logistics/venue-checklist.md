# Venue Checklist

Use this when scouting, booking, and confirming a venue for each event.

---

## Venue Scouting Criteria

| Criteria | Minimum requirement | Notes |
| --- | --- | --- |
| Capacity | 60–100 standing / 50–80 seated | Growth headroom |
| Location | Central Chandigarh / Mohali / accessible by transport | |
| Cost | Ideally free or subsidised by a sponsor | |
| AV | Projector + screen OR large display, HDMI/USB-C | |
| Microphone | At least 1 handheld or lapel mic + PA system | |
| Wi-Fi | Strong enough for 80+ concurrent users | |
| Accessibility | Step-free entrance, accessible toilets | |
| Parking | Adequate for 30–50 cars or nearby | |
| Drinks/food | Permitted to bring or serve drinks on-site | |
| Indoor | Suitable for talks (controlled noise, light) | |
| Private area | Ability to separate talk space from loud networking | Optional |

---

## Venue Booking Checklist

- [ ] Capacity confirmed for expected attendance
- [ ] Date and time slot reserved
- [ ] Written confirmation received from venue
- [ ] Cancellation/change policy confirmed
- [ ] Venue cost / cost-sharing agreement agreed
- [ ] Access time agreed (setup time before event)
- [ ] Post-event clear-up time agreed
- [ ] Venue contact name and mobile number obtained
- [ ] Key / access codes obtained (if applicable)
- [ ] Parking arrangements confirmed

---

## Day-of Venue Checklist

### AV / Tech

- [ ] Projector / screen working
- [ ] HDMI cable available (+ USB-C adapter)
- [ ] Laptop connected and displaying correctly
- [ ] Microphone working (test with audience-level volume)
- [ ] Speaker audio through PA (if applicable)
- [ ] Lighting adequate for presentations (not too bright on screen)
- [ ] Internet/Wi-Fi tested on speaker's laptop
- [ ] Wi-Fi credentials posted visibly for attendees

### Space Setup

- [ ] Seating arranged (theatre / cabaret / standing)
- [ ] Registration desk set up (badges, sign-in list, pens)
- [ ] Drinks station set up and accessible
- [ ] Signage / directional signs placed at entrance
- [ ] ProductTank banner / backdrop positioned (if available)
- [ ] Waste bins accessible
- [ ] Clear visible path to toilets and emergency exit

### Speaker Setup

- [ ] Speaker area / stage clear
- [ ] Laptop stand or lectern available
- [ ] Water provided for speakers
- [ ] Clicker / presenter available (if requested)
- [ ] Slide deck tested on venue display
