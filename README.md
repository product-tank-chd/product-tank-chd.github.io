# ProductTank Chandigarh 🚀

> A community for Product Managers and Engineering Managers/Leaders in Chandigarh to exchange ideas and experiences.

---

## About ProductTank Chandigarh

ProductTank Chandigarh brings together product and engineering professionals across the region to share knowledge, spark conversations, and build lasting connections. We cover everything that gets us excited about building products:

- **Product Design & Development** — From ideation to delivery
- **Product Management** — Strategy, roadmaps, and execution
- **Business Modelling** — Sustainable growth and value creation
- **Metrics & Analytics** — Data-driven decision making
- **User Experience** — Designing for delight and usability

We're open to companies **big and small**, and warmly welcome **early-stage technology startups** looking to learn how to get their ideas off the ground.

Events feature **guest speaker talks**, **networking sessions**, and a good old-fashioned chin-wag over a beer or two. 🍺

## Getting Started — Planning a New Event

1. Copy `events/event-planning-template.md` → rename to `events/YYYY-MM-DD-event-title.md`
2. Fill in event details, speaker slots, and logistics
3. Use `speakers/speaker-brief-template.md` to onboard each speaker
4. Schedule marketing posts using `marketing/social-media/content-calendar.md`
5. Send announcement email using `marketing/email/announcement-template.md`
6. Run the event using `events/run-of-show-template.md`
7. Send post-event recap using `marketing/email/post-event-template.md`
