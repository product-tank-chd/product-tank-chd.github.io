## Marketing Checklist

### 6+ weeks before

- [ ] Create Meetup event page
- [ ] Publish event announcement on LinkedIn, Twitter/X, Instagram
- [ ] Send email announcement to mailing list
- [ ] Reach out to speakers for bio, headshot, talk title/abstract

### 2–3 weeks before

- [ ] Post speaker spotlight #1 (social)
- [ ] Post speaker spotlight #2 (social)
- [ ] Send reminder email to mailing list
- [ ] Share event in relevant Slack/WhatsApp communities

### 1 week before

- [ ] Post countdown content (social)
- [ ] Send final reminder email (3-day reminder)
- [ ] Confirm speaker AV requirements
- [ ] Print name badges / attendee list if needed

### Day of event

- [ ] Send morning reminder notification on Meetup
- [ ] Post "See you tonight!" social story
- [ ] Brief speakers on run of show
- [ ] Live post/story during event (optional)

### After event

- [ ] Post event recap (key takeaways, photos) within 48 hrs
- [ ] Send post-event thank you email
- [ ] Thank speakers individually
- [ ] Thank sponsor(s) publicly on social
- [ ] Update past-events archive
